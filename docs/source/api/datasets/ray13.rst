ray13
==================================

.. autofunction:: eugene.datasets.ray13

Usage
----------------------------------
The ray13 function allows the `RNAcomplete dataset <http://hugheslab.ccbr.utoronto.ca/supplementary-data/RNAcompete_eukarya/>`_ from `Ray et al. <https://www.nature.com/articles/nature12311>`_ to be downloaded and extracted into a :doc:`../dataloading/dataloaders/SeqData` object.
